A modern CLI tool that automatically organizes and moves your files based on configurable categories. No manual sorting, no chaos.

⬇ Download latest ★ View on GitHub

Features

📂

Category-based rules

Define categories with extensions, filters, and destinations in a simple YAML file.

👁

Watch mode

Monitor directories in real-time and move files automatically as they appear.

↩

Undo support

Revert the last batch or any specific batch of moves from the history.

🔍

Dry-run everywhere

--dry-run on move, watch, and undo — preview before committing.

⚔️

Conflict strategies

rename, overwrite, skip, or hash_check — per category, your choice.

🔎

Powerful filters

Regex, glob, ignore patterns, age filters, and size filters per category.

🌐

Wildcard extension

Use extensions: [all] to match any file type and group by real extension.

🔄

Self-update

Stay on the latest release with movelooper self-update.

📋

Interactive setup

Generate your config file with movelooper init -i or from a template.

Installation

# 1. Download the binary for your platform from the releases page
#    https://github.com/lucasassuncao/movelooper/releases

# 2. Extract and add to your PATH, then run:
movelooper init -i
git clone https://github.com/lucasassuncao/movelooper.git
cd movelooper
go build -o movelooper .
./movelooper init -i

Quick start

Generate a config with the interactive wizard, then run.

movelooper init -i        # interactive wizard
movelooper --dry-run      # preview what would be moved
movelooper                # move files
movelooper watch          # real-time monitoring

Configuration example

configuration:
  output: both
  log-level: info
  watch-delay: 5m
  history-limit: 50

categories:
  - name: "images"
    extensions: [jpg, jpeg, png, gif, webp]
    source: "C:\\Users\\you\\Downloads"
    destination: "C:\\Users\\you\\Images"
    conflict-strategy: rename
    group-by-extension: true
    filter:
      ignore: ["screenshot_*"]
      min-age: 10m

  - name: "everything-else"
    extensions: [all]     # matches any file
    source: "C:\\Users\\you\\Downloads"
    destination: "C:\\Users\\you\\Sorted"
    conflict-strategy: hash_check
    group-by-extension: true

Commands

Command Description
movelooperRun the move operation once across all categories
movelooper watchMonitor source directories and move files in real-time
movelooper undoRevert the last batch of moves
movelooper undo --listList all recorded batches
movelooper undo --dry-runPreview what would be restored
movelooper init -iLaunch the interactive setup wizard
movelooper init -t fullGenerate config from a template
movelooper config validateValidate the config file without moving files
movelooper config showPrint the active config with all defaults resolved
movelooper self-updateDownload and install the latest release

## ðŸ§  How It Works

`movelooper` reads your configuration file (`movelooper.yaml`) or (`conf/movelooper.yaml`), scans all extensions listed in configuration file and moves matching files from the source to the destination, organizing them into folders by extension.

## Config file example (movelooper.yaml)
```yaml
configuration:
  output: both                       # console, log, file, both
  log-file: C:\logs\movelooper.log
  log-level: info                    # trace, debug, info, warn, error, fatal
  show-caller: false

categories:
  - name: images
    source: C:\Users\johndoe\Downloads
    destination: C:\Users\johndoe\images
    extensions: [jpg, jpeg, png, gif, bmp, webp]

  - name: videos
    source: C:\Users\johndoe\Downloads
    destination: C:\Users\johndoe\Downloads\videos
    extensions: [mp4, avi, mkv, mov, wmv]
```
